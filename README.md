# Chai 🍵

A minimal, terminal-based AI chat app written in Rust.
Chai is designed to give you **direct, simple access to AI models from your CLI** without getting in your way.

---

## Philosophy

Chai was created from my own experience with current AI tools, tightly integrated tools like copilot feel too intrusive, overwhelming and bloated.
But at the same time having to copy code from the text editor then paste it into some web chat ui sucks.
So here's the tools I created to solve those problems:

Chai is **non-intrusive**:
 - The AI is here to provide knowledge, **not write your code for you**.
 - You stay in control, the terminal is your environment, Chai integrates with it.
 - Use your favorite CLI tools to pipe data in and out, combining the power of AI with your existing workflow.

If any of this resonates with you, please give Chai a try.

## Features (WIP)

 - Supports multiple AI providers (OpenAI, ~~Gemini~~, ~~Anthropic~~) # Not quite yet
 - Local SQLite database to store conversations and message trees
 - Ability to easily switch models and branch conversations
 - Designed for piping input/output for seamless CLI integration
